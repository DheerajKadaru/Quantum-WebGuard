About WebGuard

Understanding the hybrid detection architecture behind Quantum WebGuard

What is Quantum WebGuard?

Quantum WebGuard is a cybersecurity-focused URL analysis platform that detects malicious links using hybrid intelligence. It combines statistical machine learning with hand-crafted threat detection rules to provide robust, explainable verdicts on URL safety.

The system is designed as a mini-project demonstration of how associative detection — correlating signals from multiple independent classifiers — can outperform either approach alone.

Technology Stack

Machine Learning Layer

  • TF-IDF Vectorization Character-level n-grams (3–6) extract lexical URL features without domain knowledge
  • Linear SVM Classifier High-speed decision boundary for binary classification with probability calibration
  • Training Data Trained on a curated dataset of safe and malicious URLs with stratified splitting

Rule-Based Detection

  • Phishing Detection Keywords like "login", "verify", "account" in suspicious contexts
  • Typosquatting & Spoofing Detects visual look-alike domains (e.g., paypa1, g00gle, micr0soft)
  • DGA-Style Domains Entropy and consonant-clustering analysis for algorithmically generated hostnames
  • Obfuscation Patterns Hex encoding, @ symbols, excessive slashes, and IP-based hosting

Hybrid Fusion Logic

  • Associative Correlation Both ML and rule outputs are combined — a malicious flag from either triggers the final verdict
  • Risk Score Computation Maximum of rule-based risk and ML confidence, capped at 100
  • Attack Classification Labels threats as Phishing, Malware, Spoofing, DGA, Obfuscation, or Suspicious Pattern
